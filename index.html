Descriptions

These apps were built with HTML, CSS, JavaScript, jQuery, PHP, and MySQL.


Login

Functions: login, change password, lost password reset. Admin functions: register a member, edit member permissions.


Database interface

This is a user interface to a database. The app presents a table view of records with viewing options. Also, user may add or edit a record. Delete record function removed.


Form validator

This app validates user inputs on a form. It validates client-side inputs through JavaScript and also validates on server by PHP. The submit button has been programmed with php mail function to send completed forms to an email address for collection.


Tag cloud

This app arranges and displays words from a selected source increasing word size based on frequency of that word.


Login 2

The code for this login app was strictly organized by MVP and uses OOP principles which are evident in the source code shown on GitHub. Please see "Source code".

About me

I've worked at each stage of the software development life cycle from planning, development, testing, product delivery, and support. My industry background is in Cable, Telecom, and Satellite service providers. I worked within B/OSS (business and operational support systems) as well as integration with external vendor applications. As a web developer, I build full stack applications with HTML, CSS, PHP, and SQL. I also work with JavaScript though limited.


Approach to work

I start with people. We talk about business needs and look at the situation. Our talks may result in requirements that drive the use cases that I write. I aim to facilitate the business process and make it less error-prone. I am guided by business logic and I refer to design patterns in my approach to projects. I follow standard organizing principles in writing code and care to make it easy to review.
